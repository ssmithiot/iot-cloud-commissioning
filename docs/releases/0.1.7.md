# Edge UI 0.1.7

Date: 2026-07-18

## Scope

- Edge UI: updated.
- Edge agent: unchanged; existing gateways may continue to report `0.1.6`.
- Cloud: records and displays the independent UI version as `0.1.7`.
- Gateway site data, IP addresses, credentials, `start.sh`, and agent
  configuration: preserved.

## Release source

`tools/releases/gw006-edge-ui-0.1.7-code.tar.gz`

`requirements.txt`, `README.md`, and `templates/` only.

## Delivered behavior

- Three-column Live Devices view with RPM-20 behavior retained.
- Multi-state value/input/output state-text write dropdown.
- Gateway-local Control Basic MVP; its program logic controls offline handling.
- No controller-online pre-write check was added to the Control Basic engine.
- Cloud dashboard shows separate `Agent <version>` and `UI <version>` values.

## Validation and rollout
